## Laravel 9 Perfect MultiLanguage app example

Full example of multilanguage app by url (domain.com/en/home, domain.com/ru/home), including storing/removing languages and CRUID of multilanguage Post Model. Static text translations stored in JSON files (lang/en.json, lang/ru.json).

### Let`s do it step by step

1) Install Laravel:
**composer create-project laravel/laravel example-app**

2) Create database and set up .env file:
DB_DATABASE=localization
DB_USERNAME=root
DB_PASSWORD=

3) Install Laravel Breeze:
php artisan breeze:install
npm install
npm run dev

4) Add new user in DatabaseSeeder
User::create([
    'name' => 'admin',
    'email' => 'admin@mail.ru',
    'password' => bcrypt('12345')
]);
